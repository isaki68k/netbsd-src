How to adapt your audio hardware driver to AUDIO2 framework.

  • Remove drain interface.

    drain'ing is done by upper layer from AUDIO2, so hardware driver no longer requires it. Be careful not to forget to remove variables which was used only here.

  • Remove mappage interface.

    mmap'ping is done by upper layer since 8.0, so hardware driver no longer requires it.

  • Remove setfd interface.

    Nobody use setfd.

  • Change query_encoding into query_format and
    change set_params into set_format.

    If your hardware supports SLINEAR_NE (Native Endian).

  • round_blocksize, round_buffersize を見直す。 同値を返すだけの場合は削除。 値を変更して返す場合は AUDIO2 側の要求と一致するか確認する必要がある。 ただし一致しなかったケースのことは考察できていない。
  • open あたりの呼び出し順序関係を考察。
  • open のフラグは常に R/W で呼ばれるように変わったことについて考察。 Read の時と Write の時で初期化動作を変えてある構造だったりしないか注意。
